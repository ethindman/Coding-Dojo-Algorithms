Coding Dojo Algorithm Challenges

Taking the algorithms from Coding Dojo's pre-bootcamp courswork and turning them into functional examples that users can play with and learn from. View on GitHub.

1. Print 1-255

Description: Write an program that can print all the numbers between 1 and 255



To make this algorithm, we can use a for loop. We set the initial value of i equal to 1, and stop the loop when i<=255. We increment i up by 1 with the syntax i++ (i.e. i = i+1).

for (i=1; i<=255; i++) {
   console.log(i);
}

2. Print Odd Numbers

Description: Write a program that can print all the odd numbers from 1 to 1,000.



To make this algorithm, we can use a for loop like we did in the first example. We set the initial value of i equal to 1 again, but stop the loop when i<=1000. We then increment i up by 2 with the syntax i+=2 (i.e. i = i+2).

for (i=1; i<=1000; i+=2) {
   console.log(i);
}

3. Print Sum of ODD Numbers

Description: Write a program that would print the sum of all the odd numbers from 1 to 10,000,000



To make this algorithm, we can use a for loop like we did in the first two examples. However, this time we need to introduce a variable, sum, and set it equal to 0. Then we set the initial value of i equal to 1 again, but stop the loop when i<=5000. We can increment i up by 2 with the syntax i+=2 (i.e. i = i+2). In order to get the sum of each iteration, we need to use the logic, "sum = sum + i" (or sum += i).

for (i=1; i<=5000; i+=2) {
    sum = sum + i;
}
console.log(sum);

4. Find Max

Description: Given an array with multiple values (e.g. [-3, 3, 5, 7]), write a program that prints the maximum number in the array. For this example, clicking the 'Run Algorithm' button will first generate a random array of 10 values between 1 and 100,000. Then another algorithm will find the largest ("max") value in that arrray.



The algorithm to find max looks like this:

x = [1,2,3,4,5];
var len = x.length;
var max = x[0];

for( i=1; i < len; i++ ) {
    if (x[i] > max) {
    max = x[i];
   }
}
console.log(max);

As you can see, we set reference value, max = x[0], or the first value of the array. As we iterate through the array, each time we compare max to the current array value x[i]. If x[i] > max, we update max so that max is equal to x[i]. In other words, each time x[i] is greater than the variable 'max', the variable 'max', will be updated. In this way, we can find the largest value in the array.
